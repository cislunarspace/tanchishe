# 游戏规则、输入控制与 UI 方案

---
status: accepted
---

游戏采用经典的网格移动规则：30×20 棋盘、每格 24 像素、窗口 720×480；蛇头每次步进一格，撞墙或撞自己即游戏结束。吃一个食物得 10 分并增长 1 段。难度分慢/中/快三档，对应基础速度 6/10/14 步/秒；长度每增加 5 段再提速 10%，但有上限。

输入方面同时支持方向键和 WASD，禁止 180 度反向掉头；每帧最多处理一次有效方向变更，使用输入队列避免高速下多输入冲突。Esc / P 暂停，R 快速重开，M 返回主菜单。

UI 使用 Bevy 原生 `bevy_ui` 实现主菜单、HUD、暂停遮罩和游戏结束界面，并加载支持中文的字体以保证菜单和分数显示正常。最高分通过 `serde` 序列化到本地 JSON 文件持久化；文件读写失败时回退到内存最高分并记录日志，不阻塞游戏。
